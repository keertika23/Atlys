

**CarouselView (SwiftUI)**

A lightweight, swipe-driven 3D carousel built in SwiftUI.


**Features**

Smooth 3D-style carousel animation
Drag gesture with snapping
Injectable images array
Scales and shifts cards dynamically
Fully SwiftUI + Combine friendly


**Usage**

CarouselView(images: ["Canada", "Tokyo", "London"])



**Project Structure**

CarouselView.swift   → Main UI component
Store.swift          → Data model + injected images
Item.swift           → Identifiable model



**Requirements**

iOS 15+
Swift 5.5+
SwiftUI
